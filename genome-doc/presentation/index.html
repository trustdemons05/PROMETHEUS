The Problem

Current methods hallucinate text

End-to-end diffusion models generate visually plausible documents —
but they invent words, scramble letters, and break layouts.

🔤

Text Hallucination

Diffusion models treat text as pixels, inventing plausible-looking but wrong characters

📐

Layout Drift

Headings become body text, margins shift, and structural hierarchy is lost

🎨

Style Loss

Original font families, ink weight, and paper texture are replaced by generic defaults

Our Approach

The Document
Genome

Instead of pixel-to-pixel restoration, we extract a structured JSON representation —
the Document Genome — then re-render from scratch with perfect fidelity.

Module 1 ✓ Complete

SIR
Style & Identity Refiner

Captures the visual DNA of a document — fonts, paper texture, ink color — as a 512-dim embedding.

🔍

Quality Patch Selector

Finds the 8 cleanest 128×128 regions via learned + handcrafted scoring

🧠

ResNet-50 Backbone

Attention-weighted pooling → single 512-dim style fingerprint

⚡

InfoNCE Contrastive Training

Hard negative mining, warmup scheduling, FP16 mixed precision

Module 2 ⟳ Next Step

DGI
Document Genome Inferrer

Reads the degraded image and outputs a complete structured Genome — every text region, its type, content, and bounding box.

🍩

Donut Backbone

OCR-free VisionEncoder → TransformerDecoder with template-guided decoding

📦

LayoutHead

Parallel MLP for bounding box regression with L1 + GIoU loss

🔧

LoRA Fine-tuning

Only 2.3% params trainable — encoder frozen, rank-16 adapters on decoder

Module 3 ⟳ Next Step

NRE
Neural Re-Rendering Engine

Combines the Genome skeleton with the SIR style embedding to generate a photorealistic clean document via controlled diffusion.

🎨

ControlNet + SD 1.5

Skeleton map as spatial conditioning, frozen SD base for generation quality

💉

Style Injection

512→768 StyleProjector generates pseudo-tokens for cross-attention conditioning

⚡

DDIM Inference

50-step denoising with classifier-free guidance for fast, high-quality output
